## Overview
A full-stack conference marketing management application built with Angular 20, Node.js, Express.js, MongoDB Atlas, and Google Gemini AI.

### Features
- Campaign Management (CRUD)
- AI Email Generator (Gemini)
- Journey Builder (CRUD)
- Responsive UI
- Cloud Deployment

## Tech Stack
- Angular 20
- TypeScript
- Bootstrap
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Google Gemini API

## Local Setup

### Backend
```bash
npm install
npm start
```

Create a `.env` file:
```env
MONGO_URI=your_mongodb_connection_string
GEMINI_API_KEY=your_gemini_api_key
```

### Frontend
```bash
npm install
ng serve
```

The application will run at http://localhost:4200

- Frontend: Vercel
- Backend: Render
